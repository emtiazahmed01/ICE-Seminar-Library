# ICE-Seminar-Library - Library Management System

A comprehensive, full-stack web application designed to digitize and streamline the library management and departmental facilities for the ICE (Information & Communication Engineering) department. This platform serves both students and administrators, offering an intuitive interface for browsing books, managing borrowings, and accessing departmental information.

✨ Features
🌐 Public-Facing Website
Home Page: A dynamic landing page featuring a slideshow of departmental activities, library opening hours, usage guidelines, and a showcase of new book arrivals.

About Page: Detailed information about the department, faculty members, vision, mission, and office staff.

Services Page: A user-friendly catalog of all available books, complete with cover images for easy browsing.

Notice Board: A dedicated page for important announcements, notices, and upcoming event details.

Forms Page: A centralized hub for all necessary departmental forms, readily available for download.

👨‍🎓 Student Dashboard
A personalized portal for students to manage their library interactions.

Features for searching and browsing the book catalog.

Functionality to view borrowing history and current issued books.

Streamlined process for requesting and managing book loans.

👨‍💼 Admin Portal
A powerful backend for librarians and administrators to have full control over the lending process.

Manage book inventory (add, update, remove).

Oversee all student borrowings and returns.

Update notices and events on the notice board.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript

Backend: PHP

Database: MySQL
